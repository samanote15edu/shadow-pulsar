# Agent Instructions

## Chat Communication Style
- Maintain a professional, concise, and clean markdown structure in all chat messages and planning summaries.
- **DO NOT** use repeated emoji borders, decorative banners, or excessive emoji walls (e.g., `--🏁--🏆--`) in your responses.
- The "Design Aesthetics" system instruction applies **exclusively** to the source code and UI of the web applications you build, NOT to your own chat output or implementation plans.
