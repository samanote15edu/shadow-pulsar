# 📝 Shadow Pulsar - Task List & Roadmap

Este archivo es el centro de control del proyecto. Aquí rastreamos la evolución del bot desde un MVP básico hasta un sistema de inteligencia avanzada.

---

## 🏗️ Evolución del Sistema (Phases)

### 🟢 Phase 1: MVP (Completada)
*   [x] **Intenciones basadas en reglas**: Comandos clave (Hola, Inventario, Ayuda).
*   [x] **Flujos estrictos**: Procesos guiados paso a paso (Onboarding).
*   [x] **UX Guiada**: El bot pregunta y el usuario responde.

### 🟢 Phase 2: Naturalidad (Completada)
*   [x] **Fuzzy Matching**: Búsqueda inteligente de productos con tolerancia a errores.
*   [x] **Entrada en lenguaje natural**: Soporte para frases de venta/surtido y fracciones naturales.
*   [x] **Refinamiento de UX**: Reducción de ruido conversacional.

### 🔴 Phase 3: Power (Pendiente)
*   [ ] **Parser Conversacional completo**: Integración de Embeddings/IA (Gemini/OpenAI).
*   [ ] **Auto-detección**: Identificación automática de intenciones y entidades sin reglas fijas.

---

## 📋 Roadmap de Funcionalidades

### 📦 Cimientos (Fase 1 Context)
*   [x] **Seguridad**: Sistema de códigos de invitación.
*   [x] **Identidad**: Captura de nombres de dueños y tiendas.
*   [x] **Comandos VIP**: Prioridad para Editar, Ayuda e Inventario.
*   [x] **Magic Links**: Dashboard dinámico sin login manual.

### 📈 Inteligencia y Gestión (Fase 2 Context)
*   [ ] **Módulo de Reportes Semanales**: Resumen automático de ventas/costos vía WhatsApp.
*   [ ] **Cierre de Caja (Corte)**: Refinamiento del comando para generar resumen de turno.
*   [ ] **Alertas de Stock Bajo**: Notificaciones proactivas de inventario crítico.
*   [ ] **Dashboard Pro**: Gráficas de tendencias y márgenes de ganancia.

### 🚀 Expansión (Fase 3 Context)
*   [ ] **Soporte Multimedia**: Fotos para registro de productos y tickets.
*   [ ] **Gestión de Fiados**: Recordatorios automáticos para clientes deudores.
*   [ ] **Sugerencias de IA**: Optimización de precios e inventario.
